Export Pinterest CSV

Click any cell to edit. Matches Pinterest official CSV format exactly.

✅ Exact Pinterest CSV format: Title, Description, Link, Image URL (link), Board, Keywords, Publish date, Note — Board must exactly match your Pinterest board name
Pinterest CSV rules:
• Board — must match your Pinterest board name exactly (capital letters matter). Example: "Summer Fashion" not "summer fashion"
• Image URL — must be a public internet link (Cloudinary/ImgBB URL). Local file paths will not work
• Keywords — space-separated words, no # symbol. Example: fashion summer outfit ideas 2025
• Publish date — format: 2025-08-15 08:00:00 or leave blank to publish immediately
• No BOM — our export is BOM-free (Pinterest rejects files with BOM)
